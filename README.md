## 📌 Project Overview

This project demonstrates **Infrastructure as Code (IaC)** using Terraform to automatically provision AWS infrastructure including VPC, Subnets, Security Groups, and EC2 instances.

> Instead of clicking in AWS Console, everything is created with code!

## 🚀 How to Use

### Prerequisites
- Terraform installed
- AWS CLI configured
- AWS account

### Steps

```bash
# Clone repo
git clone https://github.com/vaishnavipatelj/aws-terraform-infrastructure.git
cd aws-terraform-infrastructure

# Initialize Terraform
terraform init

# Preview changes
terraform plan

# Create infrastructure
terraform apply

# Destroy when done
terraform destroy
```

---

## ⚙️ Terraform Commands

| Command | Purpose |
|---|---|
| `terraform init` | Initialize providers |
| `terraform plan` | Preview changes |
| `terraform apply` | Create infrastructure |
| `terraform destroy` | Delete everything |
